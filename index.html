HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages and web applications.

History

HTML was created by Tim Berners-Lee in 1991. It has gone through many versions since then.

Usage

HTML is used to structure a web page and its content. For example, content could be structured within a set of paragraphs, a list of bullet points, or using images and data tables.
